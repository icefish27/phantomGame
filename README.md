# Phantom Game

这是一个基于 Cocos Creator 的游戏学习项目，用于学习和实践游戏开发技术。

## 技术栈

- **游戏引擎**: Cocos Creator 3.8.8
- **开发语言**: TypeScript

## 项目简介

本项目是学习 Cocos Creator 游戏开发的实践项目，通过实际编码来掌握游戏开发的核心概念和技巧。

## 快速开始

### 环境要求

- Node.js
- Cocos Creator 3.8.8 或更高版本

### 运行项目

1. 使用 Cocos Creator 打开项目
2. 点击编辑器上方的运行按钮
3. 选择目标平台进行预览

## 项目结构

```
phantomGame/
├── .creator/           # Cocos Creator 配置文件
├── settings/           # 项目设置
├── package.json        # 项目配置
├── tsconfig.json       # TypeScript 配置
└── README.md           # 项目说明文档
```

## 学习目标

- 掌握 Cocos Creator 基础操作
- 学习游戏场景搭建
- 理解组件化开发思想
- 实践游戏逻辑编写
- 学习资源管理和优化
